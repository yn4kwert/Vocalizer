This app allows to vocalize pdf-files and save it into mp3 files.

Input files should be located in '.../file_manager/input/' directory
Output files will be saved by default in '.../file_manager/output/' directory, but it could be changed by the user

The app has 3 method of saving files.
	One file - all PDF file will be saved as one .mp3 file
	Chapters - all PDF file will be splited into chapters by the word 'Глава' and each chapter will be
		       will be saved as separate .mp3 file
	RegEx separator - almost the same as 'Chapters' but separator is described by regEx written by user.
